# React Classroom Exercises and Projects

This repository is dedicated to housing all the **React exercises and projects** completed as part of my (or our) classroom curriculum.


## Project Goal

The primary goal of this repository is to track progress and consolidate learning while mastering the fundamentals of **React.js**. It serves as a portfolio of completed coursework, showcasing my (or our) development skills in front-end web development.


## Repository Structure

All exercises and projects are organized into separate folders. Each folder represents a distinct exercise or project and contains its own source code.


***Note:** To run any individual project, you will need to navigate into its specific folder.*

---

## Getting Started (Prerequisites)

To run any of the projects in this repository on your local machine, you'll need the following installed:

1.  **Node.js and npm (or Yarn):** React projects are typically managed using Node Package Manager (`npm`) or **Yarn**.
2.  A **Code Editor** (like VS Code).

---
